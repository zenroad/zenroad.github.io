免费VPN 推荐 —— ProtonVPN

Posted on 2018-10-04 | | Visitors: |

前言

我在之前的博文中提到过Protonmail这个免费的邮箱产品。ProtonMail是一个加密的webmail服务，于2013年由欧洲原子能研究组织（CERN）成员Jason Stockman、Andy Yen和Wei Sun创建。ProtonMail现由总部设在瑞士日内瓦州的Proton Technologies AG经营。ProtonMail的两个服务器分别设在瑞士的Lausanne和Attinghausen。 就我个人而言，这个邮箱兼具几大优点：

  1. 不需要绑定手机号即可使用；
  2. 没有被墙，国内国外皆可访问；
  3. 可以绑定其它邮箱实现密码找回以及新邮件提醒；
  4. 可用于申请Facebook以及Gmail；
  5. 空间相对充裕（免费版500M）。

注意！注意！注意!大陆可能直接把openvpn协议给封了，而ProtonVPN采用openvpn协议，所以在大陆的大部分地方是暂时没办法使用ProtonVPN的，期待以后的更新吧！
今天我再向各位读者介绍另一个Proton系列产品——ProtonVPN。ProtonVPN分为免费版、付费版和加强版等若干版本。
emmmmm 貌似免费版也够用了。
ProtonVPN支持平台包括：

  • macOS
  • Linux
  • Windows
  • Android
  • iOS
  • Router
    对的，居然还包括路由器～
    下面我以ubuntu16.04为例，演示如何使用ProtonVPN

申请ProtonVPN

ProtonVPN与ProtonMail可以公用一个账户或者单独分别申请，但个人建议先注册ProtonMail，原因如下：

  • 方便找回密码;
  • ProtonMail不需要手机号等等，利用ProtoMail申请VPN可以尽可能的在互联网上不留痕迹。

当然，如果您想用自己的邮箱或者手机号注册ProtonVPN也十分方便，直接访问注册地址注册即可。

注册ProtonMail

我们首先来注册一个ProtoMail账号。

  1. 在google或者百度搜索ProtoMail;
  2. 访问https://mail.protonmail.com/
  3. 选择 Create Account

  1. 接着点击free选择免费版
  2. 在新打开的页面完善信息

好了！注册成功。

注册ProtonVPN

  1. 访问如下地址 https://account.protonvpn.com/signup
  2. 选择免费版
  3. 填写邮箱信息，直接选择log in with ProtonMail account，用刚刚注册的邮箱登录。

    我们可以看到免费版具有如下功能：

其中前7天我们可以体验加强版的所有服务。
至此账号申请完毕。

配置ProtonVPN

您可以使用openvpn软件包和ProtonVPN服务器的相应配置文件为Linux设置VPN。

准备Linux VPN设置：

1.安装必要的包：

通过打开终端安装OpenVPN软件包（按Ctrl + Alt + T）并输入：

sudo apt-get install openvpn

  • 它会提示您输入密码以允许安装，输入密码以继续
  • 当它提示您确认安装时，按’y’并按“[Enter]”

注意：如果您的计算机上没有管理员权限，请与系统管理员联系并要求他们为您执行安装。

安装’network-manager-openvpn-gnome’软件包，以便更容易地使用和与Ubuntu Network Manager GUI兼容，输入：

sudo apt-get install network-manager-openvpn-gnome

  • 按[Y]，然后按[Enter]确认安装。

另外，确保已安装resolvconf：

sudo apt install resolvconf

2.获取ProtonVPN配置文件：

下载所需的配置文件:

  • 登录您的ProtonVPN仪表板，地址为 account.protonvpn.com/login
  • 选择左侧导航栏中的 Downloads
  • 找到OpenVPN配置文件部分并选择:
    • Platform: Linux
    • 协议：UDP（推荐）/ TCP（这使用端口443,如果您遇到VPN速度慢的情况请使用该协议）
    • 单击要下载的服务器的下载图标

      我们点击带free标志的图标进行下载，可选国家有日本、美国、新西兰。

      保存配置文件。

3.找到您的OpenVPN凭证

为了提高安全性，ProtonVPN设置了两个单独的凭据来验证连接。
登录ProtonVPN仪表板，单击“帐户”选项卡。 在这里，您将看到两种类型的凭据。证书ProtonVPN登录（ProtonVPN Login）在我们的应用程序中使用。OpenVPN / IKEv2用户名（ OpenVPN / IKEv2 Username）用于手动连接。 因此，请根据您的偏好配置OpenVPN凭据，因为您需要使用它们来建立Linux VPN连接。

使用VPN

方案A：使用网络管理器进行Linux VPN设置

注意该方案可能会存在DNS问题，故暂不介绍。

方案B：使用终端（CLI）进行Linux的VPN设置

注意：如果您的计算机上没有管理员权限，请与系统管理员联系并要求他们为您执行连接
打开终端（按Ctrl + Alt + T）并导航到包含配置文件的文件夹。

如果您发现使用CD命令行难以导航，则可以使用任何文件管理器打开文件所在的文件夹，然后右键单击>在终端中打开。
输入以下内容以初始化新连接：

sudo openvpn <config.ovpn>

其中<config.ovpn>是您要连接的服务器的配置文件名，例如:
de-03.protonvpn.com.udp1194.ovpn 为德国＃3服务器。
输入您的PC管理员密码以执行（openvpn将修改您的网络适配器并需要root权限）


随后，系统将提示您从步骤3获取OpenVPN凭据，输入您的凭据进行身份验证。

当您看到Initialization Sequence Completed。您已完成Linux VPN设置并成功连接到ProtonVPN服务器。

保持此终端开放，以保持与ProtonVPN的连接。 如果关闭终端，VPN连接将断开连接。
